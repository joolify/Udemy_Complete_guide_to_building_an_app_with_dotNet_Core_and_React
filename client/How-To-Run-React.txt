To start a **React app from CMD (Command Prompt)**, it depends on how the project was created. Here are the **three most common cases**, step by step.

## âœ… 1. If the project was created with **Vite** (most common now)

1. Open **CMD**
2. Go to your project folder:

```cmd

```

3. Install dependencies (only needed once):

```cmd
npm install
```

4. Start the dev server:

```cmd
npm run dev
```

5. Open the browser at the URL shown (usually):

```
http://localhost:5173
```

## âœ… 2. If the project was created with **Create React App (CRA)**

1. Open **CMD**
2. Navigate to the project folder:

```cmd

```

3. Install dependencies (only once):

```cmd
npm install
```

4. Start React:

```cmd
npm start
```

5. Browser opens automatically at:

```
http://localhost:3000
```
